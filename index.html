The internet is a global movement. It has removed all the barriers to society integrating and working together. Location is no longer a factor in who you work for or you you socialize with. Instant communication has torn down most of the limitations to global collaboration, except if you actually want to see the person your talking to. Video conferencing hasn't progressed much past one to one communications or closed proprietary solutions for small. Main Screen intends to break down this last barrier to true collaboration at a global scale

Want one click meeting set up? So do we. Want more then 10 people in a web conference? So do we. Want to record your session for those who couldn't make it live? So do we. Want the option to host it behind a firewall? So do we. Want it to work on any modern browser on any modern operating system? So do we.
